Auth

POST /api/auth/signup

Register a new user.

Request Body:
- displayName: string (required)
- email: string (required)
- password: string (required)

Response:
- JSON object with a success message

Sample Input:
{
    "displayName": "John Doe",
    "email": "john.doe@example.com",
    "password": "password123"
}

Sample Output:
{
    "message": "User registered successfully"
}

POST /api/auth/login

Login an existing user.

Request Body:
- email: string (required)
- password: string (required)

Response:
- JSON object containing accessToken, displayName, and userId

Sample Input:
{
    "email": "john.doe@example.com",
    "password": "password123"
}

Sample Output:
{
    "accessToken": "abcdef123456",
    "displayName": "John Doe",
    "userId": "user-1234"
}

League

POST /api/league/create

Create a new league.

Request Body:
- leagueName: string (required)
- fundingLimit: number (required)

Response:
- JSON object with a success message and leagueId

Sample Input:
{
    "leagueName": "Fantasy League",
    "fundingLimit": 100000
}

Sample Output:
{
    "message": "League created successfully",
    "leagueId": "league-1234"
}

POST /api/league/delete

Request Body:

Response:
- JSON object with a success message

Sample Input:

}

Sample Output:
{
    "message": "League deleted successfully"
